## my-game

一些小游戏的集合。


```bash
├─ packages                       
    ├─ canvas-rose                # canvas绘制的玫瑰
    ├─ canvas-drawing-board       # canvas画板

    ├─ game-2048                  # 2048游戏
    ├─ game-puzzle                # 拼图游戏
    ├─ game-solitaire             # 蜘蛛纸牌游戏
    └─ game-whiteblock            # 别踩白块游戏
```
